GridStep Support

GridStep is an AUv3 MIDI step sequencer for iOS.

Using GridStep

GridStep is designed to be used as an Audio Unit plugin inside a compatible host such as AUM, Drambo, Logic Pro, or GarageBand.

Quick start

1. Open your Audio Unit host.
2. Add GridStep as an AUv3 MIDI effect.
3. Add a synth on another track.
4. Route MIDI from GridStep to the synth.
5. Press play in the host.
