Terrarum GridStep support

GridStep

GridStep is an AUv3 MIDI step sequencer for iOS. Use it inside an Audio Unit host such as AUM, Drambo, Logic Pro, or GarageBand.

Quick start

Basic setup in a host:

  1. Open your Audio Unit host.
  2. Add GridStep as an AUv3 MIDI effect.
  3. Add a synth on another track.
  4. Route MIDI from GridStep to the synth.
  5. Press play in the host.
Tip: If you do not hear notes, check MIDI routing first. Most issues are host routing, not the plugin.

Common questions

Preset behavior varies by host. Some hosts do not share preset state across other hosts.

  • My preset does not load in another host: this is usually a host limitation, not GridStep.
  • No sound: GridStep outputs MIDI. You need a synth and correct routing.
  • Timing issues: confirm the host transport is running and the plugin is synced to host clock.
