MIDI Sequencer

GridStep

AUv3 MIDI step sequencer for iOS with deterministic playback and host-synced routing.

Get Support Back to Apps

Quick Start

  1. Open your Audio Unit host.
  2. Add GridStep as an AUv3 MIDI effect.
  3. Add a synth on another track.
  4. Route MIDI from GridStep to the synth.
  5. Press play in the host transport.
Tip: If there is no sound, verify MIDI routing first.

Usage Notes

Plugin type
MIDI only (no audio output)
Best hosts
AUM, Drambo, Logic Pro, GarageBand
Clock
Follow host transport and tempo
Preset behavior
Can vary by host implementation

Common Questions

  • No sound: GridStep outputs MIDI only. Route it to a synth.
  • Timing feels off: Confirm host transport is running and synced.
  • Preset differs in another host: Preset persistence is host-specific in some workflows.
