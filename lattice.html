Lattice

AUv3 MIDI instrument for iOS built around grid traversal pattern generation.

Get Support Back to Apps

Quick Start

  1. Open your Audio Unit host.
  2. Add Lattice as an AUv3 MIDI instrument or MIDI effect.
  3. Route Lattice MIDI output to a synth.
  4. Press play in the host transport.
Note: Lattice will not create sound until connected to a synth.

Usage Notes

Plugin type
MIDI only
Pattern control
Grid traversal and randomization controls
Routing
Host MIDI routing required
Transport
Runs in sync with host clock

Common Questions

  • No sound: Route Lattice to a synth track.
  • Pattern changes unexpectedly: Review Random controls and host automation.
  • Saved state differs between hosts: Host implementations may store plugin state differently.
